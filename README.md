# Лабораторна робота 4
## Розробка клієнтської частини вебдодатку

### Мета
- Закріпити навички роботи з DOM у JavaScript
- Навчитися взаємодіяти з публічним REST API
- Створити прості інтерактивні додатки
- Ознайомитися з варіантами підключення бібліотек в JavaScript

### Структура проекту
- `index.html` - головна сторінка з навігацією
- `task1.html` - сторінка для завдання 1 (робота з JSONPlaceholder API)
- `task2.html` - сторінка для завдання 2 (Todo App)
- `js/`
  - `task1.js` - функціонал для завдання 1
  - `task2.js` - функціонал для завдання 2
- `styles/`
  - `main.css` - стилі для всього проекту

### Використані технології
- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API для роботи з REST API
- Бібліотека Notiflix для сповіщень

### Завдання 1: Робота з публічним REST API
Реалізовано веб-сторінку для взаємодії з JSONPlaceholder API, яка дозволяє:
- Отримувати дані з різних колекцій API (posts, comments, albums, photos, todos, users)
- Завантажувати певну кількість елементів за раз
- Реалізована пагінація для навігації по великим наборам даних

### Завдання 2: Todo App
Реалізовано простий менеджер завдань, який дозволяє:
- Додавати нові завдання
- Помічати завдання як виконані
- Видаляти завдання
- Фільтрувати завдання за статусом (всі, активні, завершені)
- Взаємодіяти з API для збереження даних (JSONPlaceholder)

### Як запустити проект
1. Клонувати репозиторій
2. Відкрити файл `index.html` у браузері
